Modelo

El modulo Modelo es el encargado de interactuar con la base de datos, generar la conexion y las request que pida el controlador.

class modelo.Modelo(mi_database, nombre_tabla)[fuente]

La clase Modelo se construye pasandole dos parametros, la ruta de la base de datos y el nombre de la tabla (este se puede crear o cargar si ya existe). Esta clase se llama desde el controlador, en el constructor de su clase, para que pueda ejecutar las instacias del modelo y asi interactuar con la base de datos.

Parámetros:

  • mi_database (str) – Ruta de acceso a la base de datos

  • nombre_tabla (str) – Nombre de la tabla a la cual acceder o crear

__init__(mi_database, nombre_tabla)[fuente]

En el conructor de la clase generaremos la conexion con la base de datos y con la instancia _crear_tabla crearemos la tabla si es que esta no existe. Ademas asignaremos las variables de clase:

  • self.con: La conexion a la base de datos.

  • self.cursor: El cursor para actuar sobre la abse de datos.

  • self.tabla: El nombre de la tabla asignado a una variable de clase.

_crear_tabla()[fuente]

Metodo de instancia interno del modulo que se utiliza para crear la tabla en el caso de que esta no exista.

insert_row(data_querry)[fuente]

Inserta una fila completa en la tabla con los datos proporcionados.

Parámetros:

data_querry (tuple) – Tupla que contiene en orden los datos de cada fila de la tabla que se quieren ingresar.

delete_row(iid)[fuente]

Se elimina la fila indicada con el iid correspondiente.

Parámetros:

iid (int) – Valor clave de la fila que se desea eliminar.

buscar(data_querry)[fuente]

Funcion que devuelve los valores de la tabla que coincidan con los elementos pasados como argumentos.

Parámetros:

data_querry (tuple) – Tupla que contiene en orden los datos de busqueda.

Devuelve:

Todos los datos que coincidan con los pasados en el data_querry

leer_tabla()[fuente]

Selecciona todos los elementos de la tabla y los devuelve ordenados de manera descendente por la ultima fecha de modificacicon de cada uno.

leer_fila(iid)[fuente]

Retorna los datos de la fila que coincida con el id proporcionado.

Parámetros:

iid (int) – Valor clave de la fila que se desea extraer los datos.

Devuelve:

Se devuelve una tupla con los datos de la fila solicitada

update(data_querry)[fuente]

Actualiza toda una fila de la tabla.

Parámetros:

data_querry (tuple) – Tupla que contiene los datos de la fila que se quiere modificar

extract_categoria()[fuente]

Extrae de los elementos de la columna CATEGORIA sin repertir.

Devuelve:

Devuele una lista con las categorias que la tabla posee, sin repetir.
