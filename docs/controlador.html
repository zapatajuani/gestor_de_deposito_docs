Controlador

Modulo principal que se encarga de gestionar el programa, desde aqui se invocan las otras clases (Modeloy Vista), y se realiza toda la logica de la aplicacion.

class controlador.Controlador[fuente]

La clase Controlador va a ser la que se encargue, con sus instancia, de llevar este control de la aplicacion. Desde su constructor va a utilizar a las otras dos clases (Vista y Modelo), y mediante sus instancias generara la logica para poder operar la aplicaion.

__init__()[fuente]

Como se dijo anteriormenete, desde el constructor se invocaran a las otras dos clases y se inicializarade forma correcta la aplicacion.

alta()[fuente]

Se da de alta los datos que proporciona el ususario hacia la base de datos.

baja()[fuente]

Se dan de baja los datos que el usuario seleccione de la base de datos.

refresh_table()[fuente]

Se actualiza la tabla. Es una instancia interna para poder visualizar los cambios cadavez que se realizan.

actualizar()[fuente]

Se actualizan los datos en la base de datos que el usuario haya modificado.

limpiar()[fuente]

Pone todos los valores de los campos de toma de datos en un valor de default.

consulta()[fuente]

Setea los valores de los campos de entrada segun el valor de la tabla seleccionado.

buscar()[fuente]

Actualiza la tabla segun los valores que coincidan con los ingresados en los campos de entrada.

genid()[fuente]

Proporciona un valor de Item-ID con un valor aleatoreo teniendo en cuenta el resto de valores ingresado por el usuario.

exportar()[fuente]

Exporta la tabla principal a un formato .xlsx hacia alguna de las siguientes ubicaicones:

  • Documentos

  • Escritorio

  • Ubicacion del programa

Este es el orden que intentara de manera predeterminada.
